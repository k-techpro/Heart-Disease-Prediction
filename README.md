# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a patient is likely to have heart disease using machine learning techniques.  
It uses clinical parameters from a heart disease dataset and applies a Random Forest Classifier to perform binary classification.

## 🎯 Objective
To build a machine learning model that can help in the early prediction of heart disease based on patient medical data.

## 📊 Dataset
The dataset contains medical attributes related to patients, such as:

- Age
- Sex
- Chest pain type
- Resting blood pressure
- Cholesterol
- Fasting blood sugar
- Resting ECG
- Maximum heart rate achieved
- Exercise induced angina
- Oldpeak
- Slope
- Number of major vessels
- Thal
- Target

**Target column:**
- `0` = No heart disease
- `1` = Heart disease present
